AI Chat — describe a game, download it

Describe the game you want and the AI will generate a complete Godot 4 project. A Download button appears when it's ready — click to save the zip, or Open in Godot to launch it in the Web Editor.

Tip: the AI is instructed to output project files in a fenced block. If it forgets, ask "Please give me the full project as a single zip-ready JSON block." Projects are generated with SVG assets only (the Godot Web Editor currently hangs when importing zips with PNG files).

Powered by Ollama Cloud. API key stays on the server via the /api/chat proxy.
