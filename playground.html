GDScript Playground

Write code on the left, press Run (or Ctrl+Enter). A subset of GDScript runs directly in your browser — no Godot required.

script.gd
Output
What the playground supports: var/const, types (accepted, not enforced), if/elif/else, for, while, break, continue, func, return, arrays, dictionaries, Vector2, basic math, print, printerr, len, range, randi, randf, lerp, clamp, etc. A _ready() function is called automatically.

Not supported: scenes, nodes, signals, extends, class_name, await, match, Node API calls. Use real Godot for those.

See the GDScript reference for every language feature.
