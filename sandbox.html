Godot Sandbox

The real Godot engine running in your browser (WebAssembly). Create scenes, add nodes, write GDScript and press Play — no install required. This is the official Godot Web Editor.

Example projects — one click to open

Click Open in sandbox on any card: the project zip downloads and a new tab opens with the Godot launcher. On that launcher page, find the field "Preload project ZIP", pick the downloaded zip, and click Start Godot editor. A dialog here will walk you through it.

Tips

Save your work

Before closing the tab, use Project → Tools → Download project to keep your project as a zip.

Back to project list

In the editor: Project → Quit to Project List.

Run the scene

Press F5. If asked, pick the current scene as the main scene.

Web editor limits

No GDExtension / C# / plugin add-ons in the web build. Use desktop Godot for those.

Open "Hello 2D" in the sandbox

✅ Step 1/4 — Downloaded: hello-2d.zip is in your Downloads folder.

✅ Step 2/4 — A new tab just opened with the Godot launcher page (editor.godotengine.org). Switch to that tab to continue. If a popup was blocked, click the button below.

Step 3/4 — On the launcher page, load your zip:

  1. Scroll to the field labelled "Preload project ZIP".
  2. Click Choose File next to it and pick the downloaded hello-2d.zip.
  3. Click the big "Start Godot editor" button. Wait ~20 seconds.
  4. When the Install Project dialog appears, keep the default folder name, click Create Folder, then Install & Edit.

Step 4/4 — Play: Once the editor is open, press F5. If asked for a main scene, pick the .tscn file in the res:// folder.

Important: the Preload project ZIP field is on the launcher page — the first page you see. It is not inside the Project Manager. Do not use the Import button there; it has a bug with zipped projects.
