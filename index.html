What is Godot?

Godot is a free and open-source game engine used to make 2D and 3D games. It uses a scene-tree architecture: every game is a tree of Nodes. Each node has a job — draw a sprite, play a sound, detect collisions, run code — and nodes combine into scenes that you reuse and instance.

How to read this site

  1. Start with Tutorials if you are new — they teach the core ideas in order.
  2. Use Nodes like a dictionary. Jump to a node, copy the example, paste into Godot.
  3. Use GDScript when you want to learn the language itself.
  4. Use Playground to test a small idea without opening Godot.

Quick example — a moving player

Create a CharacterBody2D node, attach this script, and you have a top-down player.

extends CharacterBody2D

@export var speed: float = 200.0

func _physics_process(delta: float) -> void:
    var input_dir := Input.get_vector("ui_left", "ui_right", "ui_up", "ui_down")
    velocity = input_dir * speed
    move_and_slide()

Every part of this example — extends, @export, _physics_process, Input.get_vector, move_and_slide — is explained on this site.
